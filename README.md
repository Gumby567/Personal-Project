# Online Bookstore - Personal Project 
 
## User Stories 
 
1. As a visitor, I can browse all available books so that I can find something I want to buy. 
 
2. As a logged-in user, I can add a book to my shopping cart so that I can continue shopping before placing my order. 
 
3. As a logged-in user, I can confirm my cart and place an order so that the books are reserved for me. 
 
4. As an admin, I can add, edit, and remove books from the catalogue so that the store always shows up-to-date products. 
 
5. As a logged-in user, I can view my past orders so that I can keep track of what I have purchased.
